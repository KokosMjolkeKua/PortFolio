Title:
	Hello Health App

Description:
	
	An Interactive Health Application. Simple use of Python to store the user input, and output an f-string with a greeting and

	a welcome to the program. This projects goal is to demonstrate the proper use of repository structure, documentation and 

	virtual environment setup. 

Installation Instructions: (Using git Bash)
	
	1. Cloning the repository - (git clone https://github.com/oslometdata1300/oblig1-python-setup-hello-health-app-KokosMjolkeKua)
	
	2. Creating a virtual environment - (python -m venv venv)
	
	3. Activating the virtual environment - (venv/Scripts/activate)
	
	4. Running the application - (python hello_health.py)

Requirements:
	
	- Python 3.10 or Higher
	
	- No external dependencies for Obligation 1
