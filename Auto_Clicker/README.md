# AUTO CLICKER

# Motivation:
    This program was made because i was playing a game called cookie clicker, where the general goal 
    is to click on a cookie as many times as possible in the hopes of achieving cookie heavin. 
    It is an idle game where the main focus is to get points (cookies)
    The program is made with the sole purpose of automating the clicking required to get as many "cookies"
    as possible. 

# How To Use?
    TO activate the program you have to run it in the background
    I am currently using VSCode, with no extensions
    When it is running, to activate and deactivate you press "t"
    Be careful to not activate it while roaming the computer, since a 
    big amounts of clicks in the wrong program/computer can cause it 
    to crash.
